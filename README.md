# WEATHER-APP

# ⛅ Weather App

A simple **Weather Web App** built with **HTML, CSS, and JavaScript**. Users can enter a city name to fetch real-time weather details like temperature and description. Displays error messages if the city is not found.

## ✨ Features
- 🌍 Search weather by city name
- 🌡️ Display temperature and conditions
- ⚠️ Error handling for invalid cities
- ⚡ Lightweight and runs directly in the browser

## 🚀 Usage
1. Clone the repository.
2. Open `index.html` in your browser.
3. Enter a city name and click **Get Weather**.

## 📂 Project Structure
.
├── index.html # Main HTML structure
├── styles.css # Styling for the app
└── script.js # JavaScript logic (fetch weather data & update UI)
